A Wordpress plugin to calculate bet from Rapid API odds

== Description ==

Bet Calculator System

A Wordpress plugin to calculate bets from Rapid API's odds and let users make their bets

This Plugin connecting to Rapid API to get the bet's.

Rapid API is collecting and serving odds from several bet platforms.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bet-calculator-system` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings page and enter your Rapid API credits

== Frequently Asked Questions ==

= How is it working? =

This module is working with connecting to Rapid API with your credits.
Collect information with bets site and let users make bets.
